## Gerador de Senhas

Gerador de senhas aleatórias rodando no terminal, feito totalmente em Python.

## Funcionalidades

- Gera senhas com o número de caracteres escolhido pelo usuário
- Opção de incluir números na senha
- Validação de entradas inválidas
- Animação de loading durante a geração

## Como usar

1. Execute o arquivo `gerador.py`
2. Informe a quantidade de caracteres desejada (mínimo 5)
3. Escolha se deseja números na senha (S/N)
4. Aguarde a geração e copie sua senha

## Exemplo

Quantos caracteres? Minimo de 5: `10`
Deseja numeros na senha? S/N: `s`
Gerando sua senha... 🔐
Sua senha gerada foi: `aB3xK9mQpL`

## Tecnologias

- Python 3
- Bibliotecas: `random`, `string`, `time`
