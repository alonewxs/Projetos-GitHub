## Calculadora de Terminal

Calculadora simples que roda no terminal, feita em Python.

## Funcionalidades

- Soma, subtração, divisão e multiplicação
- Validação de operador inválido
- Validação de entrada  (impede letras no lugar de números )

## Como usar

1. Execute o arquivo `calculadora.py`
2. Escolha o operador: `+` `-` `/` `x`
3. Digite o primeiro número
4. Digite o segundo número
5. O resultado será exibido no terminal

## Exemplo

Escolha: + - / x: `+`
Digite o primeiro numero: `10`
Digite o segundo numero: `5`
`10` + `5` = `15.00`

## Tecnologias Utilizada

- Python 3
