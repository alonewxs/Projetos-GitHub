# Painel de Noticias

> Terminal de notícias em tempo real direto no seu terminal.

---

## Demo

```
==== SEJA BEM VINDO(A) AO PAINEL DE NOTICIAS ===

Escolha uma materia que deseja: tecnologia

---------
Notícia 1
---------
Autor: Kyle Torpey
Emissora: Gizmodo.com
Link URL: https://gizmodo.com/...
---------

Deseja ver a descrição da noticia? S / N: s
Qual noticia deseja ver a descrição? Digite o numero da noticia: 1

> Descrição completa da notícia...
```

---

## Funcionalidades

- Busca notícias por qualquer tema em tempo real
- Exibe as 5 principais notícias do tema escolhido
- Mostra autor, emissora e link de cada notícia
- Permite ver a descrição completa de qualquer notícia
- Tratamento de erros para entradas inválidas
