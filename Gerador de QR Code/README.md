## Gerador de QR Code

Gerador de QR Code simples que roda no terminal, feito em Python.

## Funcionalidades

- Transforma qualquer texto ou URL em QR Code
- Permite escolher o nome do arquivo gerado
- Salva o QR Code como imagem `.png`

## Como usar

1. Execute o arquivo `gerador_qrcode.py`
2. Digite o conteúdo que deseja transformar em QR Code
3. Digite o nome do arquivo de saída
4. A imagem será salva na pasta do projeto

## Exemplo

```
Seja bem vindo(a) ao gerador de QR Code
O que você gostaria de transformar em QR CODE? https://github.com/seu-usuario
Qual nome do arquivo do QR CODE? meu_qrcode
```

O arquivo `nome escolhido qr code.png` será salvo na pasta do projeto.

## Tecnologias Utilizadas

- Python 3
- qrcode
- Pillow
